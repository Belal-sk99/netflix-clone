# Netflix Clone

A clone of the [Netflix](https://www.netflix.com/) landing experience, built with vanilla HTML, CSS, JavaScript, and Tailwind CSS. Features light/dark mode toggling and interactive UI enhancements beyond the original design.

> **Axsos Academy** — Web Fundamentals Group Project | Group 1

## Features

- 🎨 **Light / Dark Mode** — Theme toggle on every page with persistent state via `localStorage`
- 📱 **Responsive Design** — Mobile-first layout using Tailwind CSS utility classes
- ⚡ **Interactive UI** — Scroll animations, hover effects, and dynamic content powered by vanilla JS
- 🔗 **Multi-Page Navigation** — Seamless HTML linking across all pages

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- [Tailwind CSS](https://tailwindcss.com/) (CDN)

## Getting Started

### Prerequisites

Any modern browser — no build tools required.

### Run Locally

```bash
git clone https://github.com/Belal-sk99/netflix-clone.git
cd netflix-clone
```

Open `index.html` in your browser, or use a local server:

```bash
# Using VS Code Live Server extension (recommended)
```

Then visit `http://localhost:3000`.

## Deployment

Deployed on [Vercel](https://vercel.com/) as a static site — every push to `main` triggers an automatic deployment.
